# 📰 Fake News Detection System

A simple and effective Fake News Detection System that utilizes Python for its core logic and data processing, and a React-based frontend for the user interface. This project aims to classify news articles or text snippets as either "Fake" or "Real" based on a provided dataset.

## 📁 Repository Structure

The repository consists of the following key files:

* **`FakeNewsDetector.jsx`**: The React component that serves as the frontend user interface. It allows users to input news text and displays the prediction results.
* **`fake_news_detector.py`**: The Python backend script. This typically handles data preprocessing, training the machine learning model (e.g., using scikit-learn), and making predictions on whether a given news text is real or fake.
* **`fakenews.csv`**: The dataset used to train and test the machine learning model. It contains historical news data labeled as fake or true.

## 🛠️ Technologies Used

* **Frontend:** React.js, JavaScript (JSX), CSS/Tailwind (if styled)
* **Backend / Machine Learning:** Python, Pandas, Scikit-learn, NLTK (Natural Language Toolkit)
* **Data Format:** CSV
